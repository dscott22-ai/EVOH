Introduction

HTTP, the Hyper Text Protocol, is the language of the web. It allows browsers and servers to communicate and has evolved froma simple tet exchange to a high-peed secure data protocol that powers almost all digital activity today.

Key Terms

  • HTTP: HypeTransfer Protocol - Enables communication between browser and server
  • Client: The browser or device requesting a resource.
  • Server: A machine that sends requested data (like a webpage).
  • Request: A call sent by a client to a server.
